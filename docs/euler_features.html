Features

Here are the main features of Euler.

  • Real, complex and interval scalars and matrices,
  • a programming language, with local variables, default values for parameters, variable parameter number, function parameters,
  • two and three dimensional graphs,
  • marker plots,
  • density and contour plots,
  • animations and stereographic plots,
  • numerical integration (Romberg, Simpson, Gauﬂ) and differentiation,
  • statistical functions and tests,
  • differential equations,
  • interval methods with guaranteed inclusions,
  • function minimizers (Brent, Nelder-Mean),
  • simplex algorithm,
  • interpolation and approximation,
  • finding roots of polynomials,
  • calling of functions in external DLLs,
  • sound playing,
  • fast Fourier transform (FFT),
  • exact scalar products using a long accumulator,
  • Postscript graphics export (thanks to Eric Boucharé)
  • editable command window with comments,
  • normal, user friendly GUI,
  • graphics export in various formats (WMF over the clipboard),
  • internal and external editor,
  • symbolic computation using Yacas.

and lots more.
