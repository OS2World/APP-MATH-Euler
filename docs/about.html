EULER

EULER is a program for quick and interactive computations with real and complex numbers, matrices, or with intervals. EULER can draw functions in two and three dimensions. It contains a programming language.

In general, EULER is a numerical matrix system. It is not a MatLab clone, but very similar to this program.

Euler comes with Yacas, a computer algebra system. So you can combine fast numerical computations with symbolic computations.
